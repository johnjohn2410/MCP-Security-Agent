MCP Security Hardening Toolkit

An open-source toolkit designed to harden Model Context Protocol (MCP) workflows. It brings together scanners, trust management, schema validation, and prompt-injection defenses into a single CLI—helping developers and researchers close critical security gaps in emerging MCP ecosystems.

Security Features

MCP Security Hardening

Defensive-by-default configuration validation, schema enforcement, and sandboxing for MCP servers with enterprise-grade security policies.

Trust & Provenance

Cryptographic verification of MCP servers, signed attestations, and mandatory trust checks with allow/deny lists for complete server validation.

AI-Powered Threat Detection

Advanced prompt injection classification, response sanitization, and payload validation with ML-based anomaly detection for real-time protection.

Comprehensive Scanning

Detect vulnerabilities across code, dependencies, secrets, and configurations with AI-powered analysis and prioritization.

MCP Integration

Seamless integration with AI assistants through the Model Context Protocol, providing secure tool access and dynamic discovery.

Enterprise Compliance

Built with enterprise-grade features including audit logging, data privacy controls, SBOM generation, and compliance reporting.

Installation

npm (Recommended)

For Node.js/TypeScript projects

npm install -g mcp-security-agent

Docker

For containerized environments

docker run ghcr.io/johnjohn2410/mcp-security-agent scan .

Binary

For CI/CD and non-Node environments

# Download from GitHub Releases
./mcp-security-agent scan .

Security Demo

See the MCP Security Agent in action:

  1. Install globally: npm install -g mcp-security-agent
  2. Apply MCP hardened policy: mcp-security-agent mcp --policy mcp-hardened
  3. Scan for vulnerabilities: mcp-security-agent scan ./your-project --type comprehensive
  4. Manage trusted servers: mcp-security-agent trust --add server.com --pubkey KEY --sha256 DIGEST --version 1.0.0
  5. Classify prompt injection: mcp-security-agent mcp --classify "ignore previous instructions"
  6. Generate compliance report: mcp-security-agent mcp --compliance

Output: Comprehensive security with MCP hardening, trust verification, AI threat detection, and compliance reporting.

MCP Security Hardening Capabilities

Configuration Validation

MCP-aware security checks for authentication, TLS, CORS/CSP, rate limiting, quotas, sandboxing, and stdio security with enterprise-grade policy enforcement.

Schema Compliance

JSON Schema validation for all MCP envelopes, protocol drift prevention, payload size limits, and JSON-only mode enforcement for secure communication.

Trust Verification

Cryptographic signature verification, public key pinning, allow/deny lists, and mandatory trust checks with SLSA/cosign attestation support.

Threat Detection

AI-powered prompt injection classification, response sanitization, suspicious directive detection, and ML-based anomaly detection for real-time protection.

Sandboxing & Isolation

Container and process isolation, resource limits, read-only filesystems, capability dropping, and seccomp/AppArmor profiles for secure execution.

Compliance & Governance

Enterprise compliance reporting, audit trails, SBOM generation, VEX documents, and comprehensive security posture assessment for regulatory requirements.
