Installation

npm (Recommended)

For Node.js/TypeScript projects

npm install -g mcp-security-agent

Docker

For containerized environments

docker run ghcr.io/johnjohn2410/mcp-security-agent scan .

Binary

For CI/CD and non-Node environments

# Download from GitHub Releases
./mcp-security-agent scan .

Quick Demo

See the MCP Security Agent in action:

  1. Install globally: npm install -g mcp-security-agent
  2. Scan a directory: mcp-security-agent scan ./your-project --type comprehensive
  3. Generate HTML report: mcp-security-agent scan ./your-project --format html
  4. Start MCP server for AI integration: mcp-security-agent server

Output: The scanner will detect vulnerabilities like SQL injection, XSS, hardcoded secrets, and configuration issues, providing detailed remediation guidance.
